--------------------------------------------------
Project Overview
--------------------------------------------------
This project is a personal portfolio website developed for the CS67 Web Technologies module. 
The website presents coursework evidence and demonstrates practical skills in HTML, CSS, 
and JavaScript.

The portfolio has been designed as a single-page website with interactive features and 
responsive design, suitable for academic assessment.

--------------------------------------------------
How to Run the Website
--------------------------------------------------
1. Download and extract the project folder.
2. Open the file named "index.html" in any modern web browser 
   (Google Chrome, Microsoft Edge, or Mozilla Firefox recommended).
3. No server or additional software is required.

--------------------------------------------------
Technologies Used
--------------------------------------------------
- HTML5 (semantic structure)
- CSS3 (styling and responsive design)
- JavaScript (client-side interactivity)

--------------------------------------------------
Features
--------------------------------------------------
- Semantic HTML structure
- Responsive layout for desktop and mobile devices
- Navigation menu with mobile toggle
- Expandable accordion for coursework evidence
- Live search filtering for evidence items
- Contact form with client-side validation
- Accessibility considerations (labels, skip link, ARIA attributes)

--------------------------------------------------
Development Environment
--------------------------------------------------
The website was developed using Visual Studio Code and tested locally 
in modern web browsers.

--------------------------------------------------
Additional Notes
--------------------------------------------------
This website is intended for coursework submission and academic review.
No external libraries or frameworks were used.
